# Требования:
`python 3.8+`
`numpy 2.3`
`matplotlib 3.10`

### Как использовать:
* В функцию function вписываем нужное нам уравнение соответствующее синтаксису питона
* При вызове solve в параметрах прописываем, чему должно быть равно уравнение (target_y) и диапазон в котором ищется решение (start, stop, step)
